There is a organizer folder which contains admin panel code.

Screenshot folder contains all the images of different functionalities.

Steps to run project

1) Download or clone the project.
2) move the entire directory to the server home directory( eg for WAMP move to www folder).
3) create database in phpmyadmin.
4) import event.sql file provided in the project directory.
5) Project is ready to launch.
